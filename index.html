Projects


Gitlet







Gitlet is a miniature model of the well-known version control system: Git. It has many of the commonly used local functionalities of Git such as commit, checkout, branch, merge, rebase, etc. This project was coded in Java and reinforces concepts of trees, file manipulation, and serialization.

Sliding Block Puzzle



Written in Java, this program solves sliding block puzzles and was built to maximize efficiency and speed. Given an initial file with the starting coordinates of all the blocks and a goal file with the ending, it utilizes the breadth first search algorithm along with the Manhattan heuristic to print out the set of corresponding moves that solve the puzzle.

Word Clock



Based off of scottbez1's Instructable titled "Sleek word clock", this was my first ever project, introducing me to the Arduino, soldering, and prototyping/building circuits. The clock contains a barebones self-built "Arduino" board with an ATMega microcontroller, shift registers, and driver ICs to keep track of time and light up the correct LEDs.
